# Alsaeh.bh - Public Project Website

This repository contains the public website for **Alsaeh.bh - Tourism Recommender System for Bahrain**, a University of Bahrain senior project.

The website introduces the project idea, objectives, main features, technologies, screenshots placeholders, team members, supervisor, and academic information.

## Project Overview

Alsaeh.bh is an AI-powered tourism recommender system for Bahrain. The system helps users generate personalized tourism plans based on their preferences, budget, trip duration, interests, and travel style.

The main system uses:

- Next.js for the frontend
- FastAPI for the backend
- Supabase PostgreSQL for the database
- Supabase Auth and Google Auth for authentication
- Gemini API for itinerary generation and chatbot responses

## Website Structure

The public website includes the following sections:

- Hero section
- About the project
- Project objectives
- System features
- Technologies used
- System architecture overview
- Screenshot placeholders
- Team members
- Supervisor information
- University and academic year information

## How to Run

This website is built as a single static HTML file.

To run it locally:

1. Open the project folder.
2. Open `index.html` directly in the browser.

Or use VS Code Live Server:

1. Install the Live Server extension.
2. Right-click `index.html`.
3. Select `Open with Live Server`.
